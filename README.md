# Simple Chat
+ An example project to demonstrate the communication between server and client application

## Instructions
+ Open the project in Visual Studio
+ Right click on "SimpleChat" in the Solution Explorer and hit "Build"
+ A new folder "x64" appears. Inside of it you find the folder "Debug" with the binaries
+ Execute the application twice. Once for servers, the other for the client.

## Remarks
+ This project currently works only under windows.
+ This project does not get any optimization in the future and will be replaced by an other more optimized project later
